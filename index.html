Evolutionary-Computing-IDE by axel22

Integrated development environment for creation and evaluation of evolutionary algorithms.

An IDE written in Java used to develop, visualize, test and benchmark evolutionary algorithms. The IDE is fully modular, allowing new algorithms to be added as plugins. It can be used to develop other algorithms as well.

Dependencies

Ant for building.

Install

For a non-windows OS replace the SWT jar files in the lib directory with SWT libraries appropriate for that OS. Then run ant to build the project. Alternatively, download the rar file for Windows, and replace the SWT jars if necessary.
